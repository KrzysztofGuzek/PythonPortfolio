Text to Morse Code Converter

Overview

This is a simple Python project that converts user-inputted text into Morse code.
It utilizes a predefined dictionary (morse_code) to map alphanumeric characters and common symbols to their Morse code equivalents.

Features

Converts uppercase letters, numbers, and select special characters into Morse code.
Identifies and lists characters that are not supported by the Morse code dictionary.

How It Works

The user is prompted to input text to convert.
The program converts each character in the text to its Morse code equivalent.
Unsupported characters are reported at the end.
